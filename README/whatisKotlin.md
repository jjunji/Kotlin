## **Kotlin**

###**Kotlin**

구글 I/O 에서 코틀린(Kotlin)이 안드로이드 어플리케이션 공식 개발 언어로 발표됨.
따라서 안드로이드 어플리케이션은 자바와 코틀린을 같이 사용하여 개발할 수 있다.
코틀린으로 작성된 소스 코드는 컴파일러에 의해 JVM에서 실행되는 자바 바이트 코드로 생성되므로 자바와 100% 호환된다.

***

###**특징**

* JVM에서 실행되므로 자바와 완전히 호환.
*  자바처럼 정적 타입의 언어(모든 표현식의 타입을 컴파일러 시점에서 알수 있다)
* 타입 추론 기능 -> 변수의 타입을 명시적으로 지정하지 않아도 처리해줌
*  객체지향 프로그래밍을 지원. 기존 클래스로부터 상속받지 않고도 해당 클래스의 기능을 추가할 수 있는 확장함수와 확장 속성을 사용할 수 있다.
*  함수형 프로그래밍을 지원. 
*  Null 값 처리에 안전.
*  문법과 코드가 간결.
* Spring5 에서 공식 지원

***

###**FP**

* 함수형 프로그래밍이란 OOP와 전혀 다른 개념이 아니다.
	* OOP를 잘 이해하고 있어야 FP를 잘 사용 할 수 있다.
*  Immutable한 자료구조를 사용해 SideEffect를 최소화.
*  상태를 바꾸기 보다는 변경 불가능한 데이터와 순수 함수를 이용한 프로그래밍.
*  모든 것은 객체다. (함수도 객체)
